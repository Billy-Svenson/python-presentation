Welcome to Python

Python is a versatile, high-level programming language known for its simplicity and readability.

- Developed by Guido van Rossum in 1991
- Widely used in web development, data analysis, AI, and more.

Key Features of Python

- Simple and easy to learn
- Interpreted and dynamic
- Extensive standard library
- Cross-platform compatibility
- Open-source with a strong community

Installing Python

1. Download from https://python.org
2. Install on your system
3. Use `python --version` to verify installation.

Hello, World!

Writing your first Python program:
```python
print('Hello, World!')
```
Output:
```
Hello, World!
```

Python Data Types

Common data types:
- int: 10
- float: 3.14
- str: 'Hello'
- list: [1, 2, 3]
- dict: {'key': 'value'}

Example:
```python
x = [1, 'Python', 3.14]
```

Control Flow

Python control flow includes:
- `if`, `else`, `elif`
- Loops: `for`, `while`

Example:
```python
for i in range(5):
    print(i)
```

Functions in Python

Defining and using functions:
```python
def greet(name):
    return f'Hello, {name}!'

print(greet('Alice'))
```

Working with Modules

Using built-in modules:
```python
import math
print(math.sqrt(16))
```

Creating your own module:
```python
# mymodule.py
def add(x, y):
    return x + y
```

Object-Oriented Programming

Key concepts in Python OOP:
- Classes and Objects
- Inheritance
- Polymorphism

Example:
```python
class Animal:
    def speak(self):
        return 'I am an animal'

class Dog(Animal):
    def speak(self):
        return 'Woof!'
```

File Handling

Reading and writing files:
```python
with open('file.txt', 'w') as f:
    f.write('Hello, File!')

with open('file.txt', 'r') as f:
    print(f.read())
```

Python Libraries

Popular libraries:
- NumPy: Numerical computing
- Pandas: Data manipulation
- Matplotlib: Data visualization
- Flask/Django: Web development

Error and Exception Handling

Handling errors using try-except:
```python
try:
    result = 10 / 0
except ZeroDivisionError:
    print('Cannot divide by zero!')
```

Introduction to Python Virtual Environments

Creating isolated environments:
```bash
# Create a virtual environment
python -m venv myenv

# Activate
source myenv/bin/activate  # macOS/Linux
myenv\Scripts\activate   # Windows
```

Python for Web Development

Frameworks like Flask and Django enable you to build web applications.

Example (Flask):
```python
from flask import Flask
app = Flask(__name__)

@app.route('/')
def home():
    return 'Hello, Flask!'

app.run()
```
